# EEG Signal Processing & Analysis System

An end-to-end EEG Signal Processing and Analysis System for preprocessing EEG signals, extracting frequency-domain features, visualizing brain activity, and performing machine learning-based mental health analysis.

## Overview

This project provides a complete pipeline for EEG data analysis, including:

- EEG signal preprocessing
- Noise and artifact removal
- Frequency band decomposition
- Feature extraction
- Statistical analysis
- Machine learning model training
- Data visualization
- Performance evaluation

The system is designed to simplify EEG signal analysis for research and educational purposes.

---

## Features

- EEG data loading and preprocessing
- Signal filtering
- Feature extraction
- Frequency band analysis
  - Delta (0.5–4 Hz)
  - Theta (4–8 Hz)
  - Alpha (8–13 Hz)
  - Beta (13–30 Hz)
  - Gamma (30–100 Hz)
- Feature visualization
- Machine Learning classification
- Model evaluation
- Interactive notebooks
- PDF documentation

## Installation

Clone the repository

```bash
git clone https://github.com/adibaba20/EEG-Signal-Processing-Analysis-System.git
```

Move into the project directory

```bash
cd EEG-Signal-Processing-Analysis-System
```

Install dependencies

```bash
pip install -r requirements.txt
```
